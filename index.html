Week 1 Summary

 

Learning Objective: By the end of this week, students will have demonstrated the following: How to use an IDE to create a JavaScript file, how to run JavaScript files with NodeJS, explain some common uses for JavaScript. 

 

Prior to Class During Class Following Class
  • Attend MAD&D Orientation.
  • Install and configure Bootcamp on their Macbook Pro.
 The learning activities for this week will include:
  • Lectures about programming concepts
  • Installation of NodeJS
  • Installation of Brackets IDE
  • Accessing CodeSchool.com
  • Setting up CodePen account
  • Learn about variables 
  • Running JavaScript programs
  • Read all the content from this week's module
  • Review all Tutorial material (video & text)
  • Read the Summary for next week
  • Complete Installs & Accounts
  • Complete JavaScript Road Trip Part 1 Level 1
