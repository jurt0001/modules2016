    Introduction to JavaScript and Basic Syntax

    JavaScript Basics

    JavaScript is a programming language which was initially designed to run within a browser and add interactivity to web pages. It allowed you to show and hide things, validate forms, and control user keyboard input. 20 years later it is used in many different locations and for many different purposes. It has become one of the most widely used and in demand languages in the world.

     

    Looking at this list you can see that it is number 3 in the world. (If that's not enough for you - Python, Ruby, and C++ are the only ones on this list that you won't be learning in depth in the MAD&D program).

    So, what does Javascript look like?

    You will see lots of these characters - ( ), { }, [ ]. They always show up in pairs.

    Every command will end with a ;

    You will also be using a thing known as dot notation. It works like this:

    World.Canada.Ottawa.Algonquin.mad9014.students["steve"].talk("Hello Class");

    See the period between each of the objects? That is how you move from one object to the next. What this code is saying is: There is an object called World. Inside that object is another object called Canada. It contains and object called Ottawa, which contains Algonquin. The Algonquin object has something called mad9014. mad9014 contains a list called students. Inside the list called students there is an item called "steve". "steve" has a method called talk( ). The round parentheses means that this is an action that the object can take. We are passing the String "Hello Class" to the talk( ) method so that the "steve" object can use it in some way.

    Now all those objects and methods are just made up by me.

    There are many objects which already exist inside Javascript. Javascript has a limited vocabulary. It only knows certain words. When you talk to the Javascript interpreter you have to use the words that it understands.

    Here is an example of some words that Javascript does understand.

    console.log("Hello class");

    Use Brackets to create a new blank file. Type the above line into the file and save it as hello.js . All Javascript files end with a .js file extension.

    Javascript understands this line. Javascript knows about the console object. The console is the Terminal window. The log( ) method will write things out to the Terminal window. Whatever you write inside the parentheses, as long as it is wrapped inside a set of quotation marks, will be written out to the terminal window when you run this script.

    Testing your Scripts

    For the first few weeks we will be using NodeJS exclusively for running and testing our Javascript. To do this we will have both Brackets and the Terminal / Command Prompt running (depending on whether you are on Mac or Windows).

    In Brackets we will create our Javascript file and save it. Eg: app.js. For now, create a folder called mad9014 and save your Javascript files inside that folder. It can be in your Documents folder or your Desktop. Later on we will be installing a web server and saving our files there.

    In the Terminal we will navigate to our folder where we saved our .js file. 

    cd - stands for Change Directory

    cd ~/Documents/mad9014/

    This command would take me to a folder called mad9014 inside the Documents folder which is inside my user folder on OSX. On Windows I would use the command

    cd C:\users\griffis\Documents\mad9014\

    to reach a similar location.

    The ls command on OSX means LiSt the contents of the current directory. pwd means Present Working Directory - it will give you the full path to your current folder.

    On Windows dir is the Directory command. It does the same thing as ls on OSX. cwd is the Windows equivalent to pwd. It stands for Current Working Directory.

    Once we have reached the correct folder/directory then we can use NodeJS to run our script

    node app.js

    Just type node followed by a space and then the name of the Javascript file.

     

    Variables and DataTypes

    Introduction to variables

    Variables are a common construct in programming languages that are designed to allow you to save values for later use.

    You can think of a variable as an empty box with a label on it. The label becomes the name you use to refer to the box and its contents. Creating an empty box and giving it a name is called declaring a variable.

    var email;

     

    Different types of boxes can hold different types of values. These different types of values are known as datatypes. The three basic simple datatypes in JavaScript are String, Number, and Boolean. Strings are any combination of letters, numbers, and characters wrapped in quotation marks. Booleans are True and False. Numbers are any numeric value with or without a decimal.

    When you want to put some information into your variable this is known as assigning a value to a variable.

    email = "griffis@algonquincollege.com";

    https://youtu.be/zjVFuft1O2Y (Links to an external site.)

    Expressions

    You can combine assignment and declaration into a single line of code if you want.

    var email = "griffis@algonquincollege.com";

    Sometimes you want to combine values either mathematically or by concatenating strings. String concatenation is accomplished with a plus sign.

    domain ="@" + "algonquincollege.com";
    email = "davidst" + domain;

    Any sort of concatenation or mathematical equation or use of methods are known as expressions.

    let and const

    The latest version of JavaScript also has two other keywords that can be used to declare different types of variables - let and const. We will come back to discuss these later.

    Compound Datatypes

    There are also complex datatypes in Javascript. When you save a list, a piece of HTML, or things like a Date in a variable, these are known as complex datatypes. They have more than just a value. These complex types have other properties and methods.

    https://youtu.be/RpU7NezV8PM (Links to an external site.) 

    if statements

    if statements are a common construct in most programming languages. Simply put, they are a way to let your program make decisions based on yes - no questions. 

    if( x == 5 ){  }

    The example above shows a simple if statement. Inside the parentheses is where you put the question you want the program to ask. Everything and anything that you place inside the parentheses will be boiled down to a yes or no answer.

    Javascript uses the concepts of truthy and falsey. A question may not give you an answer which is exactly TRUE or FALSE. So, there are a handful of things that are considered equal to FALSE. Everything else is considered TRUE.

    https://youtu.be/c83OjxNSPos (Links to an external site.) 

    When asking questions, in if statements, comparison operators are used between the values being compared. Greater than, less than, equal to, and so on, are the comparison operators.

    When you need to do things based on both the TRUE and FALSE conditions then you can use an if-else statement.

    if( x > 5 ){
      // the TRUE condition
    }else{
      // the FALSE condition
    }

    At other times you might want to do things based upon multiple possible answers. In these cases we use if - else if - else statements.

    if( x == 3 ){ 
      // if x is equal to 3
    }else if( x == 4 ){
       //if x is equal to 4
    }else{  
      //all other possible answers
    }

      Dot notation

      Dot notation is a way to access objects inside of other objects as well as their properties and methods.

      console.log( ) is an example you have already seen.

      Here are a couple imaginary examples of dot notation:

      Ottawa.Algonquin.mad9013.students.Dory.swim( );
      Ottawa.Algonquin.mad9013.students.Dory.speak( "Just keep swimming" );

      Each dot separates the objects as you drill down through the objects to reach their properties or methods. Each object is acting as a container for the next object in the chain. "Algonquin" exists inside of "Ottawa". "mad9013" exists as an object inside of "Algonquin".  There is a collection of objects called "students" inside the "mad9013" object... and so on.

      swim( ) and speak( ) are both methods. Methods are actions that an object can carry out. We know that they are methods because they have a set of parentheses at the end of the name.

      Arrays

      The Concept

      What is an Array? In the simplest terms, an Array is a list. Just like in real life, you can make a list of anything. A list of numbers, a list of groceries, a list of Prime numbers, a list of random words and numbers, a list of answers to a True or False quiz.

      An Array is a datatype in JavaScript. You can declare a variable and then put an Array in that variable. To access any of the values from the Array you use the variable name and then the index number of the item. The index number is the numeric place within the list. Index numbers in JavaScript Arrays begin with zero.

      var people = ['Jeff Lebowski', 'Han Solo', 'Hermoine Grainger', 'Tony Stark'];
      console.log( people[1] );  //outputs Han Solo

      When you want to add items to an Array you can use the push( ) or unshift( ) methods. push( ) adds the item to the end of the Array. unshift( ) adds the item to the start of the Array - the new item becomes the item at position zero.

      To remove an item from an Array you can use the pop( ) or shift( ) methods. pop( ) will remove the last item in the Array. shift( ) will remove the first item from the Array. All the items in the Array will then shift down by one index position.

      people.push('Peter Weyland');  //added to the end of the array
      var removed = people.pop();
      people.unshift('Peter Weyland');  //added to the start of the array
      removed = people.shift( );    //removed will hold the String 'Peter Weyland'

       

      https://youtu.be/arIhhRd1RPc (Links to an external site.) 

      When you want to remove something from the ANY position in the Array you would use the splice( ) method. The splice( ) method also allows you to inject new items into your array.

      people.splice(0, 2);   //removes the first two items. Start at index 0 and remove 2.
      people.splice(2, 1);   //remove only the item at position 2
      people.splice(2, 0, 'Xander Kane', 'Bilbo Baggins'); //add the two names at positions 2, 3 without removing any
      people.splice(3, 1, 'Gandalf the Grey'); //remove the name at position 3 and replace it with Gandalf

      If you want to find an item in an Array then you can use indexOf( ) or lastIndexOf( ). They will both search the array for the item you provide and return the index of the item in the Array. If the item is not found then it will return -1.

      If you want to sort an Array into alphabetical order then you can call the sort( ) method. The reverse( ) method will reverse the order of the items in your Array. 

      people.sort( );
      people.reverse( ); //Now the array is in reverse alphabetical order

      Objects

      An Object is the base object used to create all other objects in JavaScript. The cool thing about them is that you can create your own Objects and give them any properties or methods you want to create.

      To create a new empty Object:

      var phone = { };

      The syntax is similar to creating a new empty Array, but instead of square brackets we use curly braces.

      One similarity between Arrays and Objects is that you can think of them both as lists. 

      The difference is that Arrays are numbered lists and Objects are lists where you give labels to each item.

      var phone = {'type':'cell', 'number':'613-123-4321', 'manufacturer':'Samsung' };

      This sample has three items. The property names are type, number, and manufacturer. The values are 'cell', '613-123-4321', and 'Samsung'.

      To access the properties we can use two different syntaxes.

      var type = phone['type'];
      //OR
      var type = phone.type;

      The Array-like syntax or dot notation both work to either set or get the values.

      phone.number = '613-546-8939';
      phone['manfacturer'] = 'LG';
      var key = 'type';
      phone[key] = 'Cell';

       

      IF the label is stored in a variable, you MUST use the Array-like syntax though.

      Loops

      Another common feature in programming languages is the concept of loops. Say, for example, that you wanted to create a list of five random numbers. You could declare five variables and use the Math.random( ) method to generate the numbers and assign them to the variables. We could do this with just repeating the same line of code five times.

      var num1 = Math.random( );
      var num2 = Math.random( );
      var num3 = Math.random( );
      var num4 = Math.random( );
      var num5 = Math.random( );

      While a bit tedious, you can do this without too much effort.

      Now, imagine that you need to save a thousand random numbers, or ten thousand. That would be a LOT of code to write. Loops are a programmatic way of accomplishing a similar task in a repetitive manner with minimal code. Here is an example that would generate 100,000 random numbers and save them all in an Array.

      var nums = [ ];
      for(var i=0; i<100000; i++){
        nums.push( Math.random( ) );
      } 

      That's it. Four lines of code. 100,000 random numbers generated and saved in an Array.

      while loops

      There are two kinds of while loops in JavaScript, the while loop and the do..while loop. The difference between them is that the do..while loop will always run at least once because it runs the loop before checking to see if the while test passes. The while loop will test your condition before it runs the loop.

      var counter = 0;
      while(counter < 10){
        console.log(counter);
        counter++;
      } 

      var counter2 = 0;
      do{
        console.log(counter2);
        counter2++;
      }while(counter2 < 10);
