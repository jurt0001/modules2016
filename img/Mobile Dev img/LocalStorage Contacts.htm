    LocalStorage Contacts

    Re-submit Assignment
    • Due Nov 22 by 11:59pm
    • Points 15
    • Submitting a website url

    Description

    You will build an HTML page that displays a list of contact information saved in localStorage and a pop-up form for adding and editing contacts. Each item in the list should have a button to delete the contact. Adding a contact will save the information in localStorage. Editing an item will update the information in localStorage.

    Everytime localStorage is updated, the interface should be updated to reflect the change in localStorage.

    Layout

    Restrictions

    All your contact information must be saved as a single key in localstorage.

    You must use your Algonquin username as the key to store your information. Eg: abcd0001 

    When you app starts, check to see if the key exists, if not then create it and add yourself as the first contact.

    References

    Sample JSON data. This is just an example of the type of data that could be stored. Use these fields, but your own values.

    [{"fullname":"Han Solo","phone":"613-727-4723","email":"han.solo@falcon.com"},
    {"fullname":"Luke Skywalker","phone":"613-321-8765","email":"luke.skywalker@jedi.org"},
    {"fullname":"Kylo Ren","phone":"613-876-9400","email":"kylo.ren@firstorder.io"}] 

    You need to have a full name, phone number, and email address field for each person that you save. 

     

    Submission

    Upload your finished HTML, CSS, and JS to the edumedia server.

    Save your web app inside a folder /mad9014/contacts/index.html

    Eg: https://abcd0001.edumedia.ca/mad9014/contacts/index.html

     

    Challenges

    1. Add automatic sorting to the list of names. See the Numeric or Multi-dimensional Sorting page.

    2. Add CSS transitions to the modal window that appears when you add or edit a contact so it animates on and off the screen.
